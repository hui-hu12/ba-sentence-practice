- 韩国人中文学习者
- 老年人
- 初中级水平
- 课堂上需要大字、清楚、按钮容易点

## 难度设计

### 初级

重点：

- `把 + 东西 + 动词 + 好/完/上/下来/起来/开`
- 多使用衣服、门窗、手机、水、饭等生活词

### 中级

重点：

- `把 + 东西 + 动词 + 到/给/成/干净/清楚`
- 加入位置、对象、结果补语

### 高级

重点：

- 普通句改成把字句
- 韩语翻译成中文把字句
- 句子更长
- 包含课堂表达和真实交流场景

## 最新课堂版修改

- 隐藏课堂题库，课堂画面更集中
- 删除顶部大标题和说明卡
- 选择题选项加 A/B/C/D 标记
- 排序题显示固定前缀“请 把”，学生只排序后面的核心部分，例如“这张照片 / 发到 / 群里”
- “改写”改为“翻译”，直接显示韩语，让学生翻译成中文把字句

## 修改时必须回顾的注意点

- 每次修改前先看本 README，确认当前教学设计
- 不要把已经正确的选择题、排序题、翻译题判断逻辑改坏
- 排序题虽然只让学生排核心部分，但画面上必须显示完整把字句结构
- 排序题检查答案时，只检查学生排序的后半部分
- 翻译题必须显示韩语，不显示中文改写提示
- 修改后必须做 JS 语法检查和浏览器交互检查
